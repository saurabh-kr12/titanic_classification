# Titanic Classification Project

## Project Overview
This project is dedicated to predicting the survival of the passengers aboard the Titanic using various machine learning models. The goal is to analyze the dataset, understand the features that contributed to the survival rate, and build a predictive model.

## Dataset Description
The dataset used for this project is the Titanic dataset from Kaggle, which contains information about passengers on the Titanic. The dataset includes the following key features:

- **PassengerId**: An identification number for each passenger.
- **Pclass**: The class of the ticket (1st, 2nd, or 3rd).
- **Name**: Name of the passenger.
- **Sex**: Gender of the passenger.
- **Age**: Age of the passenger.
- **SibSp**: The number of siblings or spouses traveling with the passenger.
- **Parch**: The number of parents or children traveling with the passenger.
- **Ticket**: Ticket number.
- **Fare**: The amount of money the passenger paid for the ticket.
- **Cabin**: Cabin number.
- **Embarked**: Port of embarkation (C = Cherbourg; Q = Queenstown; S = Southampton).

## Features
The features are extracted from the dataset and pre-processed for better accuracy. Key features include:
- Demographic information (Age, Sex)
- Socioeconomic status (Pclass, Fare)
- Family connections (SibSp, Parch)

## Model Details
Several machine learning models were implemented, including:
- Logistic Regression
- Decision Trees
- Random Forest
- Support Vector Machines (SVM)

Each model is evaluated based on performance metrics like accuracy, precision, recall, and F1 score.
